Pas d'inquiétude ! Vos données n'ont pas été supprimées, et vous pouvez toujours vous authentifier pour les récupérer. Nous prévoyons de fermer totalement l'accès au mois de juillet 2017.

Un problème, une question ? Consultez les alternatives disponibles puis contactez-nous si vous ne trouvez pas de réponse.

Pourquoi fermer jupyter.openfisca.fr ?

Ce service ne peut pas être offert de manière fiable : chaque mise à jour d'OpenFisca-France risque d'être incompatible avec vos notebooks, et nous ne pouvons pas demander aux utilisateurs historiques de mettre à jour leur code en permanence.

Pour autant, l'absence de mise à jour signifie que personne ne peut bénéficier des nombreuses avancées d'OpenFisca-France, et que les nouveaux utilisateurs doivent utiliser une version dépassée.

Afin de nous focaliser sur les améliorations d'OpenFisca, nous avons décidé de laisser à chaque utilisateur la capacité de décider sur quelle version se baser pour écrire des formules.
